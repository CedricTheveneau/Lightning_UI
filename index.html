Here's some quick info regarding Lightning UI.

I started working on Lightning UI back in September of 2023, as part of my graduation project, as a Full-Stack dev. I had a pretty incredible concept I wanted to give birth to, the issue was : I have way too many ideas and I want to try out as much things as I can in a single project.

This big project was kind of a crash course for me, I learnt a lot, made lots and lots of mistakes, but in the end, I'm truly glad that I didn't give up and kept pushing. It gave me the opportunity to gain a lot of practice on things that revolve around coding but aren't immediate stuff we think about when dvelopping an app.

One example of those multiple things is Lightning UI : I've always wanted to create my own library (since then I've had ideas for several other ones), and upon working countless hours on it, it really felt exceptionnal. I gained a lot of time during the developpement process of this big project thanks to that library. All the blocks were pretty much ready, I didn't have to think about styling, I'd knew everything would work out well and look awesome immediatly (or would it ? *insert VSauce theme*). I'd only have to focus on actually developping, making sure all the data were displayed, all the features would work perfectly.

It took me quite some time to put together this library, and it's far from perfect, because it was my first one, and I'll probably be far more efficient on all the following ones. But I learnt something, and that was the most important part.


Now, back to you : I hope this library will be useful for you, and that you'll create amazing things with it. It's not complete at all, I have very basic features in it at the moment, but who knows ? Maybe someday I'll turn it into a proper full fledged library with everything we could imagine needing ?

That's not my priority so far, I still have a lot to do to complete my graduation project.

But for now, please enjoy this first version of Lightning UI.


Thank you again for your interest in this project,
